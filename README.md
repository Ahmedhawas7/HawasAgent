# OpenClaw Proactive AI Partner

This space runs the OpenClaw Agent 24/7.

## Setup
1. Go to **Settings** > **Variables and Secrets**.
2. Add your `TELEGRAM_BOT_TOKEN` and `GROQ_API_KEY`.
3. The bot will automatically start!
